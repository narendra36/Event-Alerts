# Event-Alerts

Developed in CSEC-HACKATHON.
Event-Alerts alerts about all the events and workshops organized in NITH by various clubs, to every registered user on our web application(Event-Alerts).

2. CSS

3. MySQL

4. PHP

5. AJAX

6. JS
